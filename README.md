# Apuntes Hacking

> Enciclopedia de hacking ético y OPSEC en castellano. Gratis. Sin cursitos, sin postureo, sin anuncios.

🔗 **Web:** https://apunteshacking.github.io/

## Qué es esto

37 módulos que van desde mentalidad y OPSEC básico hasta Active Directory,
hacking web, reversing, análisis de malware y mucho más. Todo escrito para
compartirse, no para venderse.

Puedes leer más sobre el porqué de este proyecto en la propia web:
[¿Por qué existe esta web?](https://apunteshacking.github.io/paginas/manifiesto/)
